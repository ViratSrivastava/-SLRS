# C++ files
### Folder Structure
```
SLRS/
├── include/
│   ├── RocketDataHub.h
│   └── RocketStage.h
├── src/
│   ├── main.cpp
│   └── RocketStage.cpp
├── Makefile
```
### Instructions to Compile and Run
1. Navigate to the RocketSimulation directory:
```sh
cd SLRS/cpp
```
2. Compile the program:
```
make
```
3. Run the program:
```
./RocketSimulation
```
